Rest Controller

  • Index returns json list of records
  • Index sets ok response code and json content type
  • Response sets given status code
  • Response defaults to 200
  • Response returns raw body when provided
  • Response encodes current data
  • Not found response sends msg with 404
  • Validation error response sends field keyed errors with 422
  • Create persists a vue style payload
  • Update accepts a valid vue style payload
  • Update rejects an invalid payload with 422
